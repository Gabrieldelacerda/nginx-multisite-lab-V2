Gabriel de la Cerda Project

This project, built by Gabriel de la Cerda, demonstrates a complete DevOps workflow running on AWS. The infrastructure is provisioned using Terraform, the server is configured with Ansible, and the application is deployed using Docker Compose.

The environment uses Nginx as a reverse proxy to serve multiple sites through virtual host configuration, allowing different domains to route to distinct content from the same server.

You can access the main site at http://54.233.211.29.

Additional virtual hosts are also available (after local hosts file configuration), including site1.local and site2.local.
